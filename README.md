# Employee Management & Analytics System

An application for managing employee data, featuring secure authentication and data visualization for HR analytics.

##  Key Features
* **Secure Login**: User authentication system to protect sensitive data.
* **Employee Management (CRUD)**: Full functionality to add, view, edit, and delete employee records.
* **Dynamic Search**: Quick search functionality by employee surname.
* **Data Visualization**:
    * **Salary Overview**: Graphical representation of average salary distributions.
    * **Years of Service (Tenure)**: Charts showing the length of employment across the workforce.
* **User-Friendly Interface**: Clean exit and navigation through the program.

##  Tech Stack
* **Language**:  Python 
* **Libraries**: Matplotlib (visualization), Statistics(Data analysis), Datetime(Time based calculations)

##  Analytics
The application includes a specialized module for data visualization, helping management understand salary trends and employee retention (years of service) at a glance.
